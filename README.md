# 🧱 BrickBox

**BrickBox** — это аддон для Blender, который позволяет процедурно создавать коробки из блоков (кирпичей) с полной параметрической настройкой.

Аддон автоматически рассчитывает размеры, количество блоков и швы между ними, обеспечивая удобный и быстрый способ генерации модульных конструкций прямо во вьюпорте.

---

## 🚀 Возможности

* 📦 Создание коробок из блоков по заданным параметрам
* ⚙️ Полностью параметрическая система (размеры ↔ количество блоков)
* 🔄 Автоматический пересчёт при изменении значений
* 🧱 Настройка размеров блока (ширина, длина, высота)
* 📏 Контроль размеров всей конструкции
* 🧵 Регулируемая толщина швов между блоками
* 🎨 Применение цвета ко всем блокам
* ✨ Bevel (скругление) для более реалистичного вида
* 📊 Отображение общего количества блоков
